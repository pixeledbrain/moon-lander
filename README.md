# Moon Lander
for [Game Off 2020](https://itch.io/jam/game-off-2020)

Theme was "Moonshot".

&nbsp;

## Info:

Platforming game where you control a moon lander.

My first game jam, and real project, so don't expect much.

## Controls:

Action | Key 1 | Key 2
:---: | :--- | :---
Thrust | Up | Space
Rotate Clockwise | Right | 
Rotate Anticlockwise | Left | 
Reset Level | R |
